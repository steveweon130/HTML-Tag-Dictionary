< a >

    Function:

    • defines a hyperlink
    • used to link from one page to another
    • href attribute is reqired.
      • href attribute is used to indicate the link's destination.
    • download attribute can be used.
      • download specifies that the target will be downloaded when a user clicks on the hyperlink.
    • coordinate attribute can be used.
      • coordinate attribute specifies the coordinates of a link.
    • charset attribute can be used.
      • charset attribute specifies the character-set of a linked document.
    • hreflang attribute can be used.
      • hreflang attribute specifies the language of the linked document
    • media attribute can be used.
      • media attribute specifies what media/device the linked document is optimized for.
    • name attribute can be used.
      • name attribute specifies the name of an anchor.
    • ping attribute can be used.
      • ping attribute specifies a space-separated list of URLs to which, when the link is followed, post requests with the body ping will be sent by the browser (in the background).
    • referrerpolicy attribute can be used.
      • referrerpolicy attribute specifies which referrer to send.
    • rel attribute can be used.
      • rel attribute specifies the relationship between the current document and the linked document.
    • rev attribute can be used.
      • rev attribute specifies the relationship between the linked document and the current document.
    • shape attribute can be used.
      • shape attribute specifies the shape of a link.
    • target attribute can be used.
      • target attribute specifies where to open the linked document.
    • type attribute can be used.
      • type attribute specifies the media type of the linked document.

    Encoding Formula:

    • <a href ="(Website URL)">(Click button) </a>

    Input/Output:

    • Input: <a href = "http://lordbyng.net/compsci/">http://lordbyng.net/compsci/ </a>
    • Output: http://lordbyng.net/compsci/

< h1 >

    Function:

    • defines the most important HTML heading
    • align attribute can be used.
      • align attribute specifies the alignment of a heading.
        • "center" value center-aligns the heading.
        • "left" value left-aligns the heading.
        • "right" value right-aligns the heading.
        • "justify"value makes the heading is justified to both margins.

    Encoding Formula:

    • <h1>(Heading) </h1>
    • <h1 align ="center">(Heading) </h1>
    • <h1 align ="left">(Heading) </h1>
    • <h1 align ="right">(Heading) </h1>
    • <h1 align ="justify">(Heading) </h1>

    Input/Output:

    • Input: Normal Text and <h1> Heading #1</h1>
    • Output: Normal Text and

      Heading #1

    • Input: Normal Text and <h1 align ="center"> Heading #1 </h1>
    • Output: Normal Text and

      Heading #1

    • Input: Normal Text and <h1 align ="left"> Heading #1 </h1>
    • Output: Normal Text and

      Heading #1

    • Input: Normal Text and <h1 align ="right"> Heading #1 </h1>
    • Output: Normal Text and

      Heading #1

    • Input: Normal Text and <h1 align ="justify"> Heading #1 </h1>
    • Output: Normal Text and

      Heading #1

< img >

    Function:

    • defines an image
    • src and alt attributes are required.
      • src attribute specifies the URL for an image.
      • alt attribute specifies an alternative text for an image, when the image cannot be displayed.
    • The image that will be inserted must be saved in the same folder with the HTML file.
    • height and width attributes can change the size of the image.
      • height specifies the height(length) of the image
      • width specifies the width of the image
      • The image would be inserted by its actual size if height attribute and width attribute are not defined.

    Encoding Formula:

    • <img src ="(Image URL)" alt ="(Image Name.[Image Type])">
    • <img src ="(Image URL)" alt ="(Image Name.[Image Type])" height ="(Specific Image Height)" width ="(Specific Image Width)">

    Input/Output:

    • Input: <img src ="X1n8LRQH_400x400.jpg" alt ="X1n8LRQH_400x400.jpg">
    • Output:
    • Input: <img src ="X1n8LRQH_400x400.jpg" alt ="X1n8LRQH_400x400.jpg" height ="40" width ="40">
    • Output:

< li >

    Function:

    • defines a list item
    • used in ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).
    • reversed, start, and type attributes can be used to change the bullet pointts' shape in <ol> tag.
      • reversed attribute specifies that the list order should be descending
        • Ex. 9,8,7,6,5...
      • start attribute specifies the start value of an ordered list.
      • type attribute specifies the kind of marker to use in the list.
        • Ex. 1,A,a,I,i...

    Encoding Formula:

    • <ol> Start of the List <li>Item #1</li> <li>Item #2</li> End of the List </ol>
    • <ol type = "1"> Start of the List <li>Item #1</li> <li>Item #2 </ol> End of the List
    • <ol type = "A"> Start of the List <li>Item #1</li> <li>Item #2 </ol> End of the List
    • <ol type = "a"> Start of the List <li>Item #1</li> <li>Item #2 </ol> End of the List
    • <ol type = "I"> Start of the List <li>Item #1</li> <li>Item #2 </ol> End of the List
    • <ol type = "i"> Start of the List <li>Item #1</li> <li>Item #2 </ol> End of the List
    • <ol start = "(Starting Number [N])"> Start of the List <li>Item #N</li> <li>Item #N+1 </ol> End of the List
    • <ol reversed > Start of the List <li>Item #2</li> <li>Item #1 </ol> End of the List
    • <ul> Start of the List <li>Item #1</li> <li>Item #2 </ul> End of the List
    • <ul type= "disc"> Start of the List <li>Item #1</li> <li>Item #2 </ul> End of the List
    • <ul type= "circle"> Start of the List <li>Item #1</li> <li>Item #2 </ul> End of the List
    • <ul type= "square"> Start of the List <li>Item #1</li> <li>Item #2 </ul> End of the List

< video >

    Function:

    • defines a video, movie, or other video streams
    • supports only .mp4, .ogg , and .webm file types.
    • The video that will be inserted must be saved in the same folder with the HTML file.
    • The videos which are directly brought from the Internet are only supported by Chrome, Internet Explorer(Edge), FireFox, Safari, and Opera browsers.
      • not supported in Internet Explorer 8 and earlier versions, versions below Opera 25, and any other browsers.
    • If video is brought from Internet, source tag is needed to be included.
      • source tag is used to specify multiple media resources for media elements.
    • src attribute is required.
      • src attribute specifies the URL for an image.
    • If video is brought from Internet, controls attribute is needed to be included.
      • controls specifies that video controls should be displayed such as a play/pause button etc.
    • If video is brought from Internet, type attribute is required.
      • type attribute specifies the Internet media type of a script.
    • height and width attributes can change the size of the video.
      • height attribute specifies the height(length) of the video
      • width attribute specifies the width of the video
    • muted attribute can be used to specify that the audio output of the video should be muted.
    • loop attribute can specify that the video will start over again, every time it is finished.
    • autoplay attribute can specify that the video will start playing as soon as it is ready.
    • preload attribute can specify if and how the author thinks the video should be loaded when the page loads.
    • If video from the Internet can be copied as embed, copy it and paste it beside <video>.

    Encoding Formula:

    • <video width="(width of the video)" height="(height of the video)" controls ><source src= "(Video name[Video Type])" type= "video/(Video Type)"> </video>

    Input/Output:

    • Input: <video> <iframe width ="560" height= "315" src ="https://www.youtube.com/embed/H4p6njjPV_o" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe></video>
    • Output:
